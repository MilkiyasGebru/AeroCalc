### This Repository includes the scripts needed to generate the across psds from a given building dimensions

Steps to generate the across psds for training our dataset.

1. Create a csv file with the headers: depth, width, and height
2. Populate the csv file with building dimensions you want to generate the across psds
3. In the proportional_dataset_generation.py file, pass the csv file
4. Execute the proportional_dataset_generation.py :

   `python3 proportional_dataset_generation.py`
